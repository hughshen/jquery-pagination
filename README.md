#用法

```
$('.wrap').pagination();
```

#参数

* `itemSelector` - 内容选择器
* `paginationSelector` - 分页容器选择器
* `showItems` - 显示内容数量
* `showRange` - 分页显示范围
* `showPrev` - 是否显示上一页
* `showNext` - 是否显示下一页
* `prevClass` - 上一页自定义类
* `nextClass` - 下一页自定义类
* `prevText` - 上一页文字
* `nextText` - 下一页文字
* `sepClass` - 分页分割自定义类
* `sepText` - 分页分割文字
* `pagesWrap` - 分页父元素
* `pagesItem` - 分页子元素
* `pagesWrapClass` - 分页父元素自定义类
* `pagesItemClass` - 分页子元素自定义类
* `pageActiveClass` - 当前分页自定义类

#说明
